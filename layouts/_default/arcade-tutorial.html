the arcade has a database

      
your game can save data that persists between plays. add a leaderboard in 2 lines:

      
net.rank("ekansh", 99)      -- save score
local top = net.top(5)            -- fetch top 5
for i, e in ipairs(top) do
  print(i .. ". " .. e.name .. "  " .. e.score)
end

each game has its own leaderboard namespace. data is stored in Redis (Upstash) via a serverless API. it persists forever.

        

        

          
net.set("lives", 3)
local lives = net.get("lives")

          
key-value storage scoped per game. good for saving player progress, unlocks, or anything that should persist. reads and writes are async — execution resumes automatically when done.
