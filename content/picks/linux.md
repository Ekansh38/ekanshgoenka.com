A phenomenal open-source operating system. Lean, lightweight, performant, and really well-made overall. I love it for desktop use, server use, and especially embedded systems. I also love learning from it, and it's beautiful that it's open-source. A great piece of engineering. Using Linux feels like holding a nicely crafted glass cube, whereas Windows feels like some brown plastic. At least Mac is Unix-based.
