GENO
Go · in progress · Bubbletea TUI · evolutionary simulation

A genetic evolution simulator that models populations under real evolutionary pressure. Organisms carry actual genome sequences that mutate, cross over, and express into phenotypes. Selection acts on fitness scores derived from the environment.

What it models

Each organism has a fixed-length genome encoded as a byte slice. Reproduction is sexual — two parent genomes recombine at a random crossover point with a configurable mutation rate per base. Fitness is evaluated each generation against an environment that can shift, simulating changing selection pressures.

Design decisions

Written in Go for performance on large populations. The TUI is built with Bubbletea, showing live population statistics, genome diversity metrics, and a generation ticker. Simulation runs in a separate goroutine; the UI subscribes to updates via channel. No genome compression — raw bytes for simplicity and speed.

Status

Core evolution loop is working. Fitness landscape and phenotype expression are placeholder. Currently building out the TUI panels for genome visualization and implementing a configurable environment model. Sexual reproduction and drift are implemented; selection pressure is next.
